# 3DGS-SLAM-Papers

Processed corpus and tooling for a 3DGS/SLAM paper collection.

This repository contains:

- batch PDF-to-markdown extraction scripts
- graph construction and refinement scripts
- categorization, deduplication, and export utilities
- generated graph outputs, reports, dashboards, and local browsing pages

Primary outputs:

- `graphify-out/filtered/recategorized/final_reviewed/final_reviewed_v2/final_reviewed_v3/deduped/graphify_final_reviewed_v3_deduped.json`
- `graphify-out/filtered/recategorized/final_reviewed/final_reviewed_v2/final_reviewed_v3/deduped/master_reports/master_dashboard.html`
- `graphify-out/filtered/recategorized/final_reviewed/final_reviewed_v2/final_reviewed_v3/deduped/master_reports/paper_browser.html`
- `graphify-out/filtered/recategorized/final_reviewed/final_reviewed_v2/final_reviewed_v3/deduped/year_trends/year_trends.html`

Notes:

- raw PDF files are excluded from Git to keep the repository pushable to GitHub
- extracted markdown, graph outputs, and generated reports are retained
